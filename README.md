# Wisteria Tree Mod 🌸

Brings the beautiful Wisteria tree to the Valley! This mod adds a fully custom, standalone tree with harvestable flowers, custom artisan tea brewing, and full in-game configuration.

---

## 🌟 Main Features

*   **Brand New Tree**: Adds a gorgeous Wisteria tree featuring vibrant purple flowers to the game.
*   **Pierre's Shop Integration**: Purchase the Wisteria Sapling directly from Pierre’s General Store for 1,000g.
*   **Harvestable Flowers**: Fully grown trees produce **1 Wisteria Flower per day** during **Spring & Summer**. Shake the tree to harvest it! (The tree sprite has flowers baked in, so no fruit overlays will clutter its beautiful canopy).
*   **Wisteria Tea**: Process a Wisteria Flower in a **Keg** (takes 3 in-game hours) to brew a soothing Wisteria Tea. Drinking it restores health/energy and grants a **+30 Max Stamina buff** for 7 hours!
*   **Fast Growth Toggle (Optional)**: Don't want to wait an entire season for your tree to grow? Toggle the **Fast Growth** option in the mod settings to make the tree advance one growth stage every day.

---

## ⚙️ Requirements

To use this mod, you will need the following frameworks installed:
*   [SMAPI](https://smapi.io/) (4.0.0+)
*   [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915) (2.0.0+)
*   [Generic Mod Config Menu (GMCM)](https://www.nexusmods.com/stardewvalley/mods/5098) *(Optional, for changing config settings in-game)*

---

## 📥 Installation

1. Download and unzip the contents of this mod.
2. Drop **both** unzipped folders directly into your Stardew Valley `Mods` directory:
    *   `[CP] Wisteria Tree` (the Content Patcher pack for textures, items, and recipes)
    *   `WisteriaTreeMod` (the C# SMAPI helper for custom fruit tree behaviors and config)
3. Launch the game using SMAPI, and head over to Pierre's!
